# Discord Bot

## Description

A basic Discord Bot built using javascript (NodeJS) and discord.js npm package.

## Usage

### Create a `.env` file

```
DISCORD_BOT_TOKEN={your bot token}
```

### Install Dependencies

```
npm i

# Or

npm install
```

### Run in dev mode

```
npm run dev
```

### Run in production

```
npm start
```
